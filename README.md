# CaverNicolas
Game made for Ludum Dare Game Jam(LD 45 5/10/2019)

# Description
A game about beating your own limits by going from the bottom to the top of your power.

# Controls
- W/A/S/D for movement
- Space/LeftClick to atack
- Escape to close game
